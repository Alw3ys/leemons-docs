# Manage Calendar Events

As an administrator, you can create global events for the programs you have created so that all related users can see them in their calendars.

![Calendar Events](/img/calendar_events.png)

:::tip WHAT ABOUT REGIONAL HOLYDAYS?
We are working on an advanced calendar configuration system, in the meantime, with this functionality you can specify the days of vacation or annual breaks for each program in one place.
:::
