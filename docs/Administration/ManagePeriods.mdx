# Manage Evaluation Periods

As an administrator, you can define customized evaluation periods for the teachers of each course to use when preparing their midterm or final grade reports.

![Evaluation periods](/img/periodos_figma.png)

:::tip WHAT ABOUT REGIONAL HOLYDAYS?
We are working on an advanced calendar configuration system, in the meantime, with this functionality you can specify the days of vacation or annual breaks for each program in one place.
:::
