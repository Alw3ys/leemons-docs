# Manage Calendar Events

As an administrator, it is possible to create global events for the programs that have been created so that all related users can see them in their calendars.

![Calendar Events](/img/calendar_events.png)

:::tip WHAT ABOUT REGIONAL HOLYDAYS?
We are working on an advanced calendar configuration system, in the meantime, with this functionality users can specify the days of vacation or annual breaks for each program in one place.
:::
