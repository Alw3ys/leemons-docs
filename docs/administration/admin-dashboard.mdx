# Admin Dashboard

The Leemons administrator dashboard makes it possible to quickly review the current status of the educational programs configured in your organization, as well as technical data about the performance of your server.

![Evaluation Systems](/img/ADMIN_DASHBOARD.png)


:::tip Color and cover for programs
To better identify your programs, add a cover image and a color using the "Learning Programs" option. The color will help to differentiate the events associated with the program you later create.
:::
